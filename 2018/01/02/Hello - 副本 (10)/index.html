Yu Writer 是什么

Yu Writer 是一款纯文本及 Markdown 文本编辑器，你可以把它当作记事本来使用，编辑一些短小的文本，或者记记事写写心情日记，你也可以用它来写技术文档、论文、自媒体文章，还可以用来写博客，甚至写小说、写书。

Yu Writer 希望用户能专注于内容创作、尽量避免其他因素干扰，所以使用了一种近似纯文本的 Markdown 文档格式来书写文档。在 Yu Writer 里你不能像其他文字处理软件一样随意更改字体的大小、颜色和样式，也不能随意对段落、图片进行排版。

但这并不是说 Yu Writer 就不能生成图文并茂、样式丰富的文档，因为通过 Markdown 你可以轻松定义文档的组成元素，比如基本的粗体、斜体、删除线以及加亮，也可以定义标题、链接、图片、列表等，甚至更复杂的如表格、计算机源代码块、数学公式、脚注等等。而且书写过程中手指几乎不需要离开键盘，当你熟练 Markdown 的基本格式之后，你就能体会到什么是奋笔疾书。因为一般文字处理软件经常需要你停下来，先用鼠标或触控板选定文字，在更改其样式，然后接着书写。特别是在使用触控屏幕输入文字的设备上，使用 Markdown 格式来书写能保持创作思路的连贯性、提高书写效率。

文档库

对于重度书写作者，平时可能会积累大量的文章，时间一长，每次写作时可能都需要先花一点时间来寻找需要的文档在哪，然后再打开、阅读、编辑、关闭。Yu Writer 希望能简化这些工作，所以内置了一个文档库用于管理你的所有创作。文档库支持全文搜索，支持 Tag 标签管理，用于快速找到你需要的文档。

你可以创建多个库，在库里还可以创建文件夹，用来细分同一个库里的文档。Yu Writer 的文档库支持把库创建在本地的任何文件夹之内，比如你可以把文档库创建在 iCloud Drive, Google Drive, OneDrive 和 Dropbox 等等云储存的文件夹里面，这样就可以实现在多台设备间同步文档了。

博客、电子书、个人知识库

Yu Writer 的文档库分为通用、博客、电子书以及个人知识库等 4 种类型，后 3 种分别用于自动生成静态博客（类似 Jekyll、Hexo）、生成 ePUB 电子书以及 Wiki，目前这 3 种都还没完成，所以这里就先不讲解了。

同时 Yu Writer 支持把文档库链接到现有的 GitHub Pages、Jekyll、Hexo、Hugo、GitBook Toolchain 等项目，你可以在 Yu Writer 里编辑这些项目里头的文档，Yu Writer 对这些项目有一定的集成支持，比如能罗列项目的图片资源，辅助插入文档间的链接，新建 Post 时会自动生成一个带日期格式的文档名称等。以后还会支持执行外部指令，调用这些项目的工具链以实现重新生成、发布等功能。

热保存、日志式版本记录

你可以随时打开应用开始写作，也可以随时关闭应用，文档是热保存的，再也不会因为忘记手动保存、误操作或者程序崩溃而丢失数据。每次打开应用时还能自动打开上次编辑时的文档以及还原光标位置，方便立即进入书写状态。

文档的编辑全过程，包括每个文字的录入和删除，都会记录下来，真正可以做到回滚到历史上任意版本、任意时刻。还支持重播观看创作的全过程哦！

为重度写作用户而做的优化

Yu Writer 有一个可编辑的大纲视图，在这里你可以更改缩进层次、可以调整段落结构，轻松组织超长篇幅的文档。同时程序支持可缩放的即时预览窗口，支持多标签、多窗口并行编辑，支持点击文档内的链接跳转到目标文档，支持从网页复制过来的素材自动转换为 Markdown 格式等等，都是为重度写作用户而优化的功能。
